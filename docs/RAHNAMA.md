## ۱. مقدمه

کیا نت (Kiya Net) یک پلتفرم خدمات اینترنتی و کافی‌نت آنلاین است که امور اداری، قضایی، دانشگاهی، مالیاتی، بانکی، خودرویی و طراحی گرافیکی را به‌صورت غیرحضوری و ۲۴ ساعته انجام می‌دهد.

### ویژگی‌های اصلی
- ثبت سفارش آنلاین با قیمت شفاف
- پرداخت از طریق کیف پول، درگاه آنلاین (زرین‌پال/پی‌پینگ/تست) یا کارت‌به‌کارت
- گاوصندوق امن مدارک
- پیگیری زنده وضعیت سفارش
- چت اختصاصی با اپراتور
- پنل مدیریت برای ادمین و اپراتورها
- ربات تلگرام و بله با دستورات /start، /orders، /wallet، /support، /miniapp، /help
- Mini App داخل تلگرام/بله با احراز هویت خودکار initData
- اتاق انتظار سرگرمی‌محور (اخبار، بازی، تخفیف)
- CMS کامل بدون کد و Live Edit برای مدیریت محتوا، منو، خدمات، کارت بانکی و شبکه کسب‌وکار

## ۲. راه‌اندازی پروژه

### ۲.۱. پیش‌نیازها
- Node.js ۱۸ یا بالاتر
- حساب GitHub
- دیتابیس PostgreSQL (Neon توصیه می‌شود برای Cloudflare)
- حساب Cloudflare (برای deploy نهایی)

### ۲.۲. مراحل نصب

```bash
# ۱. کلون کردن ریپو
git clone https://github.com/AvidKiya/KIYA-NET.git
cd KIYA-NET

# ۲. نصب وابستگی‌ها
npm install

# ۳. تنظیم متغیرهای محیطی (فقط برای توسعه محلی)
cp .env.example .env
# مقادیر DATABASE_URL و JWT_SECRET را پر کنید

# ۴. ساخت دیتابیس و جداول
npx drizzle-kit push

# ۵. Seed کردن داده‌های اولیه (ادمین، دسته‌ها، خدمات)
npx tsx src/db/seed.ts

# ۵.۱. Sync پیش‌فرض‌های CMS بعد از seed یا تغییر schema
npx tsx scripts/sync-cms.ts

# ۶. اجرای محیط توسعه
npm run dev
```

### ۲.۴. نصب اولیه (Setup Wizard)
در اولین اجرا، اگر دیتابیس خالی باشد، به‌صورت خودکار به `/setup` هدایت می‌شوید. در اینجا:
1. رشته اتصال دیتابیس را وارد کنید.
2. توکن Cloudflare R2 را وارد کنید (برای ذخیره فایل‌ها).
3. توکن ربات تلگرام را وارد کنید.
4. توکن ربات بله را وارد کنید.
5. مرچنت کد درگاه پرداخت را وارد کنید.
6. روی **«تست و راه‌اندازی»** بزنید.

> در production، توکن‌ها در جدول `system_settings` با کلیدهای `TELEGRAM_BOT_TOKEN` و `BALE_BOT_TOKEN` ذخیره می‌شوند و از پنل ادمین در تب **«ربات‌ها»** قابل ویرایش‌اند. تنظیمات درگاه پرداخت نیز در تب **«تنظیمات»** با کلیدهای `PAYMENT_GATEWAY`، `ZARRINPAL_MERCHANT` و `PAYPING_TOKEN` ذخیره می‌شود.

## ۳. راهنمای مشتری

### ۳.۱. ورود و ثبت‌نام
1. وارد صفحه **«ورود»** (`/login`) شوید.
2. یکی از روش‌های زیر را انتخاب کنید:
   - **ورود با Google**: فقط یک کلیک.
   - **ورود با تلگرام**: با لینک deep-link یا مینی‌اپ.
   - **ورود با شماره موبایل + OTP**: شماره را وارد کنید و کد ارسالی به تلگرام/پیامک را تأیید کنید.
3. پروفایل خود را تکمیل کنید (نام، کد ملی، آدرس، شبا).

### ۳.۲. مشاهده خدمات
در صفحه اصلی ۷ پیشخوان مجازی وجود دارد:
- **امور قضایی و حقوقی**: ثنا، ابلاغیه، نوبت قضایی، سوءپیشینه
- **دانشگاه و آزمون‌ها**: کنکور، انتخاب واحد، سیدا، مای مدیو
- **مالیات و اصناف**: اظهارنامه، کد اقتصادی، ثبت شرکت
- **وام و امور بانکی**: وام ازدواج، مرآت، یارانه، سجام
- **خودرو و پلیس +۱۰**: تعویض پلاک، خلافی، عوارض، طرح فروش خودرو
- **طراحی، گرافیک و چاپ**: رزومه، پاورپوینت، کارت ویزیت، بنر، تایپ
- **تامین اجتماعی و بیمه**: ثبت‌نام، سوابق، فیش حقوقی، کسر از حقوق، لیست کارگاهی

### ۳.۳. ثبت سفارش
1. روی پیشخوان موردنظر کلیک کنید یا مستقیماً در نوار جستجو سرویس را جستجو کنید.
2. سرویس موردنظر را انتخاب کنید.
3. توضیحات موردنیاز را بنویسید.
4. مدارک لازم را از **گاوصندوق مدارک** انتخاب یا آپلود کنید.
5. در صورت نیاز، گزینه **سفارش فوری** را فعال کنید (۳۰٪ هزینه اضافه).
6. روش پرداخت را انتخاب کنید:
   - **پرداخت از کیف پول**: اگر موجودی دارید.
   - **درگاه آنلاین**: نیاز به تنظیم مرچنت کد در پنل ادمین دارد.
   - **کارت‌به‌کارت**: واریز به کارت اعلامی و آپلود رسید.
7. روی **«ثبت سفارش»** بزنید. کد رهگیری به شما داده می‌شود.

### ۳.۴. پیگیری سفارش
- وارد صفحه **«سفارشات من»** (`/orders`) شوید.
- وضعیت سفارش به‌صورت زنده نمایش داده می‌شود:
  - در انتظار / بررسی / نیازمند اطلاعات / در حال انجام / تکمیل شده / لغو شده
- در صورت نیاز اپراتور از طریق چت سفارش با شما ارتباط می‌گیرد.
- بعد از تکمیل، فایل خروجی از همین صفحه قابل دانلود است.

### ۳.۵. کیف پول
- در صفحه **«کیف پول»** (`/wallet`) می‌توانید موجودی را ببینید.
- برای شارژ:
  - **درگاه آنلاین**: وارد کردن مبلغ و هدایت به درگاه.
  - **کارت‌به‌کارت**: واریز و آپلود رسید برای تأیید ادمین.
- از شارژ کیف پول برای پرداخت سریع سفارشات استفاده کنید.

### ۳.۶. گاوصندوق مدارک
- در صفحه **«مدارک من»** (`/vault`) می‌توانید مدارک خود را ذخیره کنید:
  - کارت ملی رو و پشت
  - شناسنامه
  - پایان خدمت
  - مدرک تحصیلی
  - عکس پرسنلی ۴×۳
- فایل‌ها به‌صورت امن در فضای ابری (R2) نگهداری می‌شوند.
- در زمان ثبت سفارش، به‌جای آپلود مجدد، فقط مدارک موردنیاز را از گاوصندوق انتخاب کنید.

### ۳.۷. اتاق انتظار
- **اخبار**: خواندن آخرین اخبار ایران و دنیا از منابع RSS.
- **بازی**: بازی کردن و جمع‌آوری امتیاز برای دریافت کد تخفیف.
- **درباره ما**: آشنایی با کیا نت و شبکه کسب‌وکارهای مرتبط.

### ۳.۸. محتوای قابل تنظیم (CMS)
- تمام متن‌ها، رنگ‌ها، منوها، اطلاعات تماس، کارت بانکی و شبکه کسب‌وکار از پنل ادمین قابل ویرایش است.
- مدیر کل می‌تواند با **ویرایش زنده** روی هر صفحه، بدون ورود به پنل، محتوا را تغییر دهد.

## ۴. راهنمای اپراتور (ادمین عادی)

### ۴.۱. ورود به پنل اپراتور
1. به آدرس `/admin` بروید.
2. شماره موبایل خود و رمز عبور را وارد کنید.
3. اگر توسط مدیر کل تعریف شده‌اید وارد پنل می‌شوید.
4. فقط به ماژول‌هایی که مدیر کل به شما اختصاص داده دسترسی دارید.

### ۴.۲. پذیرش و انجام سفارش
1. در تب **«سفارش‌ها»** سفارشات در انتظار را ببینید.
2. روی سفارش موردنظر **«قبول»** بزنید تا به شما تخصیص یابد.
3. مدارک مشتری را بررسی کنید.
4. در صورت نیاز به اطلاعات بیشتر، وضعیت را به **«نیازمند اطلاعات»** تغییر دهید و در چت پیام بدهید.
5. وقتی سفارش تکمیل شد، وضعیت را به **«تکمیل شده»** تغییر دهید و فایل خروجی را آپلود کنید.
6. به‌محض تکمیل، پورسانت شما به‌صورت خودکار محاسبه و به کیف پول شما واریز می‌شود.

### ۴.۳. درخواست برداشت
1. در تب **«مالی و پرداخت»** موجودی خود را ببینید.
2. مبلغ موردنظر برای برداشت را وارد کنید.
3. شماره کارت/بانک را در توضیحات بنویسید.
4. روی **«ثبت درخواست برداشت»** بزنید.
5. مدیر کل درخواست را بررسی و پرداخت می‌کند (معمولاً پنج‌شنبه ۱۵ تا ۲۱).
6. رسید پرداخت و متن دلخواه مدیر برای شما ارسال می‌شود.

## ۵. راهنمای توسعه‌دهنده

### ۵.۱. دستورات اصلی

```bash
# نصب وابستگی‌ها
npm install

# اجرای dev
npm run dev

# اعمال تغییرات دیتابیس
npx drizzle-kit push

# Seed دیتابیس
npx tsx src/db/seed.ts

# تایپ‌چک
npm run typecheck

# لینت
npm run lint

# بیلد برای Cloudflare Pages
npx @cloudflare/next-on-pages
```

### ۵.۲. متغیرهای محیطی (فقط برای dev)

فایل `.env` برای توسعه محلی:

```env
DATABASE_URL=postgresql://... (Neon PostgreSQL)
JWT_SECRET=یک رشته تصادفی قوی حداقل ۳۲ کاراکتر
```

> در production، توکن‌های ربات، درگاه پرداخت، R2 و... از پنل ادمین در `system_settings` ذخیره می‌شوند.

### ۵.۳. ساختار فایل‌های کلیدی CMS
- `src/components/cms/CMSContext.tsx` — Provider و هوک `useCMS` برای بارگذاری محتوا.
- `src/components/cms/Editable.tsx` — کامپوننت ویرایش زنده.
- `src/components/cms/LiveEditToggle.tsx` — دکمه شناور فعال‌سازی Live Edit.
- `src/components/cms/CMSAdminPanel.tsx` — پنل ۹ تب مدیریت محتوا در `/admin`.
- `src/app/api/cms/route.ts` — API CRUD و لاگ تغییرات.
- `scripts/sync-cms.ts` — همگام‌سازی پیش‌فرض‌های CMS بعد از تغییر schema.

### ۵.۴. ساختار فایل‌ها

- `src/app/` — صفحات و API routes
- `src/components/` — کامپوننت‌های UI
- `src/db/` — دیتابیس، schema و seed
- `src/lib/` — توابع کمکی، احراز هویت، فرمت
- `src/lib/bot/` — موتور ربات تلگرام/بله
- `docs/` — مستندات و راهنما

## ۶. سوالات متداول (FAQ)

**س: آیا کیا نت روی موبایل کار می‌کند؟**  
ج: بله. کیا نت به‌صورت Mobile-First و PWA طراحی شده و قابل نصب روی صفحه اصلی iOS/Android است.

**س: اطلاعات و مدارک من امن است؟**  
ج: بله. فایل‌ها در فضای ابری امن (R2) ذخیره و با TLS ۱.۳ منتقل می‌شوند.

**س: آیا می‌توانم بدون ثبت‌نام سفارش بدهم؟**  
ج: خیر. برای ثبت سفارش، احراز هویت (شماره موبایل + OTP یا ورود با Google/Telegram) الزامی است.

**س: چطور می‌توانم به ادمین/اپراتور تبدیل شوم؟**  
ج: مدیر کل باید از پنل مدیریت، شماره موبایل یا آیدی عددی تلگرام/بله شما را به‌عنوان اپراتور تعریف کند.

**س: پرداخت کارت‌به‌کارت چقدر طول می‌کشد؟**  
ج: رسید شما بعد از تأیید ادمین (معمولاً در چند ساعت تا یک روز کاری) اعمال می‌شود.
